# 💬 Community Discussion Forum with Real-Time Chat (MERN Stack)

A full-stack real-time community discussion platform built using **MongoDB, Express, React, Node.js (MERN)** and **Socket.IO**.  
Users can register, login, create discussions, and chat in real-time inside discussion rooms.

---

## 🚀 Features

- 🔐 User Registration & Login
- 🔑 JWT Authentication
- 💬 Create & Join Discussions
- ⚡ Real-Time Chat using Socket.IO
- 📩 Store chat messages in MongoDB
- 📜 View discussion history
- 🔄 Live message updates without refresh
- 🧑 Simple user-based messaging system

---

## 🛠 Tech Stack

### Frontend
- React.js
- Axios
- Socket.IO Client

### Backend
- Node.js
- Express.js
- Socket.IO

### Database
- MongoDB
- Mongoose

### Authentication
- JSON Web Token (JWT)
